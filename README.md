# kintai

Web ページ: https://misoclub.github.io/kintai

概要
----
kintai は、シンプルで軽量な勤怠（出退勤）管理向けのフロントエンド実装です。HTML、JavaScript、CSS を主体とした構成で、サーバーサイドを必要としない静的な形で動作します。小規模チームや個人の試作、プロトタイプとして利用しやすい設計を目指しています。

主な特徴
--------
- ブラウザのみで動作するシンプルな UI
- 出勤・退勤の記録（ブラウザのローカルストレージへの保存）
- 日別・月別の簡易集計表示
- 最小限の依存（HTML / JavaScript / CSS）

動作環境
--------
- 任意の近代的なウェブブラウザ（Chrome、Firefox、Edge、Safari 等）
- ローカルファイルでも動作しますが、GitHub Pages や簡易 HTTP サーバーから配信することを推奨します

クローンと起動方法
----------------
1. リポジトリをクローンします:
   ```
   git clone https://github.com/misoclub/kintai.git
   cd kintai
   ```

2. 開発中にローカルで確認するには、単純に `index.html` をブラウザで開くか、以下のように簡易サーバーを使います:

   - Python 3 を使う場合:
     ```
     python -m http.server 8000
     # ブラウザで http://localhost:8000 を開く
     ```

   - または npm の http-server (インストール済みの場合):
     ```
     npx http-server -c-1
     # 表示された URL をブラウザで開く
     ```

使い方（基本）
-------------
- ページを開き、「出勤」「退勤」ボタンで時刻を記録します。
- 記録はブラウザのローカルストレージに保存されるため、同一ブラウザで継続して利用できます。
- 月別ビューや履歴表示で、出退勤の合計や履歴を確認できます。

設定・カスタマイズ
-----------------
- 表示や保存のロジックは主に `js/` 以下のファイルにまとまっています（実際のファイル構成はリポジトリをご確認ください）。
- データ永続化をサーバー側に移す場合は、適切な REST API エンドポイントと認証を追加することで対応可能です。
